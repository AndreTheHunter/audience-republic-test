## Testing

In [watch mode](https://cljdoc.org/d/lambdaisland/kaocha/1.91.1392/doc/7-watch-mode) using [Just](https://github.com/casey/just):

```sh
just test
```

Or without watch:

```sh
lein kaocha
``` 

Or:

```sh
lein test
```

## REPL

Using [Just](https://github.com/casey/just):

```sh
just repl
```

Or:

```sh
lein repl
```
